# DashScan Web App

**Blockchain explorer for the [Dash](https://www.dash.org/) network**

Browse blocks, transactions, masternodes, wallet addresses.

---

`React` `TypeScript` `TanStack` `Tailwind CSS` `Dash UI Kit` `Shadcn` `Vite`

</div>

## Getting Started

**Prerequisites:** Node.js 20+, [pnpm](https://pnpm.io/)

```bash
pnpm install
pnpm dev
```

Dev server runs at **http://localhost:3000**

## Scripts

```bash
pnpm dev        # Start development server
pnpm build      # Production build
pnpm test       # Run tests
pnpm check      # Lint + format
pnpm lint       # Lint only
pnpm format     # Format only
```
